## 本项目使用的技术栈为 vue3 + vite + ts + pinia + element-plus + vue-router + axios +mockjs + less + eslint + tailwindcss

## 项目中使用mockjs模拟后台数据，体验真实的数据请求loading效果，在开发过程中封装了axios，方便调用接口，同时封装了全局按钮loading效果。项目中使用了权限管理知识点，根据用户不同权限来展示相应的菜单和按钮，同时使用了pinia来管理全局状态，方便组件间共享数据

## 在线演示地址： https://renwenhao-code.github.io/rwh-github.github.io/manage3/index.html#/login

## 本项目为vue3的脚手架项目，集成了vue3全家桶，以及常用的第三方库，方便快速上手开发

## 通过此项目可以学习到ts在vue3中的使用，以及vue3的新特性，如组合式api，响应式数据，生命周期钩子，组件间通讯等

## 项目启动

```
npm install
npm dev

```
